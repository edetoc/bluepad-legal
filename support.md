# bluepad Support

Need help with bluepad? Send me a message at **[e_detoc@hotmail.com](mailto:e_detoc@hotmail.com)**.

bluepad is built by an independent developer. Every message is read and answered personally — usually within a few days.

When reporting a bug, please include:

- Your iPhone model and iOS version
- Your Apple Watch model and watchOS version (if relevant)
- A short description of what you were doing
- Screenshots, if possible

— Eric Detoc
